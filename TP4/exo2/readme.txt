How to generate HTML from XML using XSLT

1.Save the XSLT code as a separate file, e.g. "menu.xsl".
2.Save the XML code as a separate file, e.g. "menu.xml".
3.Open a terminal window and navigate to the directory containing the XSLT and XML files.
4.Run the following command: xsltproc menu.xsl menu.xml > menu.html
5.Open the menu.html file in a browser to view the generated HTML.
